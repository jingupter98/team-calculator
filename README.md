# Командный проект: Калькулятор на Python

Данный репозиторий создан в рамках выполнения практического кейса по дисциплине "Программирование на Python (базовый уровень)".

## Описание проекта
Приложение представляет собой консольный калькулятор, который позволяет выполнять базовые математические расчеты. Проект разработан с использованием системы контроля версий Git для демонстрации навыков командного взаимодействия.

## Реализованный функционал
* **add(a, b)** — сложение двух чисел.
* **subtract(a, b)** — вычитание чисел.
* **multiply(a, b)** — умножение чисел.
* **divide(a, b)** — деление с защитой от деления на ноль.

## Тестирование
В проекте реализованы автоматические модульные тесты (находятся в файле `test_calculator.py`). Для запуска тестов используется встроенная библиотека `unittest`.
